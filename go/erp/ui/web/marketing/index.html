Enterprise Features

Built with modern architecture and enterprise-grade capabilities

🎯

Protobuf Data Model

72 schema files define a unified type system with cross-module references and automatic serialization.

⚡

Module Selection

Enable or disable modules at runtime with automatic dependency management and visual dependency map.

🌐

Multi-Currency Exchange

Real-time currency conversion with automatic amount recalculation when switching currencies.

📱

Desktop + Mobile Apps

Full-featured web and mobile interfaces with complete parity, floating headers, and card navigation.

🔧

Factory-Driven UI

Shared factories for forms, columns, enums, references, sections, and SVG illustrations.

🔒

Validation Framework

Declarative validation builder with required fields, enum checks, and reference verification.

🔌

API-first Design

RESTful APIs for all 376 services with consistent endpoints and Protobuf serialization.

📑

Full Audit Trail

Complete transaction history across all modules with AuditInfo on every record.

📊

Mock Data Generation

Realistic test data generators for all services with phased dependency ordering and cross-module references.

💻

Zero Frontend Dependencies

Pure vanilla JavaScript with no build step. No frameworks, no bundlers, no node_modules.

☸

Kubernetes Native

Production-ready K8s manifests with DaemonSets, StatefulSets, and virtual network overlay.

🔬

Comprehensive Tests

25 test suites covering service getters and handlers for every module.
